# atomno-mcp-pharma

**Справочник лекарств РФ прямо в AI-ассистенте** — Russian drug reference for AI
agents: check ГРЛС registration, get a drug card, look up ЖНВЛП price caps, check
recalls, and link the official instruction — straight from the state registers.
Works in Cursor, Claude Desktop, Cline, and any MCP client.

Спросите ассистента «Панадол зарегистрирован? Есть ли он в ЖНВЛП и по какой
цене?» — и получите структурированный ответ из государственных реестров
ГРЛС/ЖНВЛП (Минздрав РФ), а не ручной перебор форм на неудобном сайте. Для
аптек, дистрибьюторов, отделов закупок (44-ФЗ/223-ФЗ), фарм-юристов и
разработчиков медсофта.

> **Это справочная выдача государственных реестров, а не медицинская
> консультация.** Сервис не даёт показаний, дозировок-назначений, оценок
> совместимости и подбора замен. По вопросам применения обратитесь к врачу или
> фармацевту.

## Что умеет

- **Проверка регистрации в ГРЛС** — статус РУ (действует/отменён/приостановлен), номер и дата.
- **Карточка препарата** — ТН и МНН, формы и дозировки, держатель РУ, производитель, страна, код АТХ.
- **Поиск с мостиком МНН↔ТН** — по действующему веществу находит торговые наименования и наоборот.
- **ЖНВЛП + предельная цена** — входит ли в перечень и какова ПЗЦ по позициям.
- **Отзывы и изъятия** — письма Росздравнадзора по препарату или серии.
- **Официальная инструкция** — ссылка на первоисточник ГРЛС.

```bash
pipx install atomno-mcp-pharma
# или: uvx atomno-mcp-pharma
```

```json
{
  "mcpServers": {
    "pharma": {
      "command": "uvx",
      "args": ["atomno-mcp-pharma"],
      "env": { "MCP_PHARMA_API_KEY": "<ваш-ключ-Pro>" }

  }

```

## Переменные окружения

| Переменная | Описание | Обязательна |
|---|---|---|
| `MCP_PHARMA_API_KEY` | Ключ Pro (заголовок X-API-Key). [Получить](https://atomno-mcp.ru/pricing#pharma-pro) | да |
| `MCP_PHARMA_API_BASE` | URL hosted-бэкенда (по умолчанию — прод) | нет |
| `MCP_PHARMA_TIMEOUT` | Таймаут HTTP, сек (default 30) | нет |
| `MCP_PHARMA_LOG_LEVEL` | Уровень логирования (DEBUG/INFO/WARNING/ERROR, default WARNING) | нет |

## Тулы

| Тул | Вход | Что возвращает |
|---|---|---|
| `check_drug_registration` | name? / mnn? / ru_number? | статус РУ (действует/отменён/приостановлен), номер и дата |
| `get_drug_card` | name? / ru_number? | ТН, МНН, формы/дозировки, держатель РУ, производитель, страна, АТХ |
| `search_drug` | query, limit? | ранжированный список препаратов по ТН и МНН |
| `get_zhnvlp_price` | name? / mnn? | флаг вхождения в ЖНВЛП + ПЗЦ по позициям |
| `check_recall` | name? / series? | письма Росздравнадзора: наличие, дата, реквизиты, причина как в документе |
| `get_instruction` | name? / ru_number? | ссылка на официальную инструкцию ГРЛС (+ текст как госинформация) |

Каждый ответ содержит `source` (реестр-первоисточник), `source_date` и `disclaimer`.
